# Data assets
## NASA
[GEOS](https://gmao.gsfc.nasa.gov/geos-systems/): The Goddard Earth Observing System (GEOS) model consists of a group of model components that can be connected in a flexible manner in order to address questions related to different aspects of Earth Science.

[GMAO](https://gmao.gsfc.nasa.gov) (Global Modeling and Assimilation Office)

GMAO [Products](https://gmao.gsfc.nasa.gov/gmao-products/): GEOS FP, GEOS IT, GEOS-CF, GEOS-S2S, [MERRA-2](https://gmao.gsfc.nasa.gov/gmao-products/merra-2/), 7km-G5NR, SMAP L4
